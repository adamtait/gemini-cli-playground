# Gemini CLI Playground

A collection of skills and extensions for the [Gemini CLI](https://github.com/google-gemini/gemini-cli), providing many of the powers of Claude Code but powered by Gemini models.

## Extensions

### Ralph Wiggum

Inspired by the official [Claude Code plugin](https://github.com/anthropics/claude-code/blob/main/plugins/ralph-wiggum/README.md) of the same name, this extension has been ported to the Gemini CLI.

#### Installation

To install the Ralph Wiggum extension, run the following command from the root of this repository:

```bash
gemini extensions link extensions/ralph-wiggum
```
